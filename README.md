# FSD-Smart-Crowd-Control

A full-stack crowd control management system.

## Project Structure

- `server/`: Node.js Express server with MongoDB integration.
- `client/`: React frontend built with Vite.

## Getting Started

### Server
1. Navigate to the `server` directory.
2. Run `npm install`.
3. Run `npm start`.

### Client
1. Navigate to the `client` directory.
2. Run `npm install`.
3. Run `npm run dev`.
